In this tutorial, I will show you how you how to create our own JavaScript functions as well as utilize some built-in functions (called methods). One of the best places to learn the capabilities of JavaScript and the web is the Mozilla Development Network(MDN). It provides a plethora amount of knowledge about Web APIs and how to use mostly every built-in function in JavaScript.

Introduction to Functions

Functions are the building blocks of JavaScript. Simply put they are procedures that perform a task. But let's breakdown even further how we perceive a function.

            
              /* Lets Breakdown how to create our very first function */

              function name(parameters) {
                statements;
              }
  • function — This is a JavaScript reserved keyword used to ensure JavaScript knows we want to create a function.
  • name — This is the name we determine that describes what the function is doing.
  • parameters — There are unlimited amount of parameters or arguments* you can have in your function. These are the name of the values you pass to your function. *arguments is also a reserved word in JavaScript
  • statements — Statements are the steps or procedures your JavaScript code will run.

I know what you're thinking... that's cool but how do I use it. Let's take a look at how we can write a function to multiply three numbers together.

            
              // Previous way of multiplying numbers together
              var total = 5 * 35 * 2;

              // Step 1. Create the multiply function
              function multiply(x, y, z) {
                var total = x * y * z;
                console.log('Total: ', total);
                return total;
              }

              // Now when we call the multiply function
              multiply(5, 35, 2);
              // This will log -> Total: 350

              // How is this helpful you might think? This allows our function to be more dynamic
              // We could also call multiple iterations of our function
              multiply(1, 1, 1)
              // This will log -> Total: 1
              multiply(2, 10, 2)
              // This will log -> Total: 40

              // As a default if there is no `return` in a function it will automatically return undefined.
              // So if we want to be able to use these values we must specify a value to return

Let's build our own

Problem: A teacher tasks you, a Teaching Assistant, with dropping the lowest quiz grade and getting the average of the top two quiz scores.

Task: Create a function that drops the lowest score and returns the average of the top two quiz scores.

Hmm.. that seems like a lot. We will break it down into small chunks and use some built-in functions/methods to create our function.

          
            // This is our array of students
            var studentList = [
              {
                name: 'Bob Jones',
                grades: [75, 80, 90]
              },
              {
                name: 'Jim James',
                grades: [55, 100, 90]
              },
              {
                name: 'Eli Jones',
                grades: [75, 100, 90]
              }
            ];

            // This is the function we will use to get the student's
            // quiz average
            function getStudentQuizAverage(student) {
              // Step 1. Create variables to store student name & grades array
              // First we grab the students name
              var name = student.name;
              // Next we grab the students grades
              var grades = student.grades

              // Step 2. We happened to look on MDN under JavaScript Math and
              // found a Math.min() method that takes integer parameters
              // and finds the lowest one.
              var worstQuizGrade = Math.min(grades[0], grades[1], grades[2]);

              // Step 3. We create a new array called `newGrades` to
              // ensure we don't mess with the old grades just
              // in case the teacher changes their mind later on.
              var newGrades = [];

              // Step 4. We will loop through all of our grades and
              // if we find don't find our worstQuizGrade
              // we'll add it to our newGrades array
              for(var i = 0; i < grades.length; i++) {

                // Step 5. We will check if the grades at the index match
                // our worstQuizGrade.  If it doesn't match, we
                // will add that grade to our newGrade array using .push
                if(grades[i] != worstQuizGrade) {
                  newGrades.push(grades[i]);
                }
              }

              // Step. 6 Here we will determine our finalGrade which is just
              // The newGrades at each index divided by 2
              var finalGrade = (newGrades[0] + newGrades[1]) / 2;

              // Step 7. We print out the finalGrade to see in the console
              console.log(name + ' Quiz score Average: ' + finalGrade + '%\n')

              // Step 8. We return the finalGrade to add to our new array
              return name + ' Quiz score Average: ' + finalGrade + '%\n';
            }

            // Step 9. Now we can loop through the studentList and print out the grade
            // for each student by calling the getStudentQuizAverage
            for(var i = 0; i < studentList.length; i++) {
              getStudentQuizAverage(studentList[i])
            }
            // Output =>
            // Bob Jones Quiz score Average: 85%
            // Jim James Quiz score Average: 95%
            // Eli Jones Quiz score Average: 95%


            // ** Finally. If we want to create a new updated list of student
            // quiz grades we can first create a variable `updatedStudentList`
            // and loop through the old student list and push the values there.
            var updatedStudentList = [];
            for(var i = 0; i < studentList.length; i++) {
              // For clarity we can break this down
              // updatedList.push(value) => Pushes/Adds the value to the end of an array
              // getStudentQuizAverage(student) => Returns the `Student name Quiz Score..` string

              updatedStudentList.push(getStudentQuizAverage(studentList[i]));
            }
            // Output =>
            // Bob Jones Quiz score Average: 85%
            // Jim James Quiz score Average: 95%
            // Eli Jones Quiz score Average: 95%

            console.log(updatedStudentList);
            // Output =>
            // [Bob Jones Quiz score Average: 85%, Jim James Quiz score Average: 95%,Eli Jones Quiz score Average: 95%]
