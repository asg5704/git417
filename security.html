Web Security

Your browser has tons of information about your browser. Some of this information are used by hackers and bad actors to exploit you.

appName

It gives the name of the browser displaying the page — this could be potentially useful to a hacker as they can narrow down exploits to a single browser.

appVersion

Gives the browser version displaying the page — this is very useful as it allows a bad actor insights on which browser version you are using and possibly use an exploit if you haven't updated your browser recently.

geolocation

This gives access to the user's location based on the users permission settings — while this needs to be explicitly allowed/denied, this information could be used to target victims as it uses IP to determine latitude and longitudes of your location.

onLine

Tells if the browser has a connection to the internet — this can sometimes throw false positives in virtualization software.

platform

This tells the OS or Operating System of the computer — like the appVersion this can be used to exploit older Operating Systems like Windows XP.

width

Displays the width of the screen

Here are some tips on how to stay safe on the web

  1. Be careful of what you download

    Bad actors and cybercriminals will trick you into downloading software that will compromise your system and use your personal information for further destruction.

    https://www.pcworld.com/article/243782/five_tips_to_avoid_malware_in_mobile_apps.html

    Summary — Stay safe by never downloading from sources you don't recognize including email attachments.

  2. Choose strong passwords

    Passwords are the number one weak spot on the Internet. People tend to choose easy passwords and cybercriminals can use bruteforcing to guess passwords.

    https://staysafeonline.org/stay-safe-online/securing-key-accounts-devices/passwords-securing-accounts/

    Summary — Use unique passwords for each site. Try to create passwords with more than 15 characters. Use a password management software like LastPass to keep credentials secure.

  3. Keep your antivirus software up to date

    Antivrius software cannot protect against every threat, but it can detect and remove most malware.

    https://edu.gcfglobal.org/en/internetsafety/how-to-avoid-malware/1/

    Summary — Run antivirus and antimalware software regularly. Ensure you keep your software up-to-date especially operating systems (OS) and browser updates.
